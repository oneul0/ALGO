# Q2. Make Sum Divisible by P
## 접근
슬라이딩 윈도우로 구간 늘리면서 (전체합 - 부분합) % p 가 0이 되는 구간을 찾음

## 실패
`1 <= nums.length <= 10^5`
위 풀이는 O(n^2) 임으로 시간복잡도를 만족하지 않음

## 정답
전체합을 total, 임의의 부분연속집합의 합을 sub라 하자.
(total - sub) % p = 0
total % p - sub % p = 0
total % p = sub % p

이때, sub는 부분연속집합은 합을 다음과 같다.
sub = prefix[j] - prefix[i]

k 를 total % p 라 하면, 

(prefix[j] - prefix[i]) % p = k
(prefix[j] % p - prefix[i] % p) % p = k

prefix[j] % p - prefix[i] % p  = k
prefix[j] % p = k + prefix[i] % p 

그러나 k + prefix[i] % p 는 prefix[j] % p 보다 클 수 있으므로

prefix[j] % p  = (k + prefix[i] % p) % p

누적합을 구할 때 prefix[x]의 인덱스들을 기록해놓는다.
map[(prefix[i] % p + k) % p] 의 인덱스 중
그 차가 가장 작은 인덱스가 최소구간 이고,
i를 늘리며 최소구간을 갱신한다.

# Q3. Ways to Make a Fair Array
## 접근
접근도 못함

## 정답

i번째 원소 이후의 원소들의 홀짝은, i번째 원소 제거 이후에는
짝수였던 원소가 홀수가 되고
홀수였던 원소가 짝수가 된다.
따라서

leftEvenSum = evenSum[i]
rightEvenSum = totalSum - oddSum[i+1]

leftOddSum = oddSum[i]
rightOddSum = totalSum - evenSum[i+1]

newEvenSum = leftEvenSum + rightEvenSum
newOddSum = leftOddSum + rightOddSum

newEvenSum == newOddSum 인 경우를 세주면 된다.
