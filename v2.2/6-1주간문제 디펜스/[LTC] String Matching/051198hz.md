# Q1. Repeated Substring Pattern
## 접근
윈도우 크기를 정해 패턴이 반복되는지 확인하려 했으나 이건 너무 뻔함

## 정답
문자 P에 대해 어떤 문자열 S가

`P + P + P + P + ... + P`

로 이루어져 있다면, 1 <= K < N 에 대해
어떤 K에서 자기 자신으로 돌아오는 경우가 반드시 있다.
문자열 `S + S`.dropFirst().dropLast() 가 S를 포함한다면
S는 회전했을 때 자기 자신으로 돌아오므로, `P + ... + P` 임을 알 수 있다.

# Q3. Repeated String Match
## 접근
접근조차 못함

## 정답
어떤 문자 P에 대해, 문자열 S가 `P + ... + P` 의 부분문자열이라면 
P를 최대 ceil(S.count / P.count) + 2 번 반복한 문자열 T 내에 등장한다.

```
P = abcde
S = eabcdea

abcd"e" + "abcde" + "a"bcde
```

따라서 2 ~ ceil(S.count / P.count) + 2 까지 P를 더하며
문자가 포함될 때 반복 횟수를 반환한다.
