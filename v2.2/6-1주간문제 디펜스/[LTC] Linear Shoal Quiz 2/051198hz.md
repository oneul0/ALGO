# Q1. Remove Duplicate Letters
## 접근
문자가 처음 등장할 때에만 따로 적어둔다.

```
class Solution {
    func removeDuplicateLetters(_ s: String) -> String {
        var answer = ""
        var appearMap = [Character: Bool]()
        
        for c in Array(s) {
            if !appearMap[c, default: false] {
                appearMap[c] = true
                answer.append(c)
            }
        }
        
        return answer
    }
}
```

## 실패
문자 중복은 없으나, 사전순으로 가장 작은 문자열을 보장하지 못함

## 정답
각 문자가 등장하는 가장 마지막 인덱스를 기록하고
스택에서 문자열 앞부터 순회하며 쌓는다.
사전순 작은 단어이기 때문에, top의 글자가 현재 추가하려는 글자보다 크고, 나중에 다시 등장하는 단어라면, pop후 push한다.
