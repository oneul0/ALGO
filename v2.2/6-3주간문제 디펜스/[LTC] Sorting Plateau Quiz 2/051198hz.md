# Q1. Minimum Number of Days to Make m Bouquets
## 접근
정렬해서 풀려고 했는데, 인접한 꽃이어야 한다는 조건으로 불가.
## 실패
bloomDay = [7,7,7,7,12,7,7]
m = 2
k = 3

정렬하면

[7,7,7,7,7,7,12]

7일째에 꽃이 6개 피었으므로

6 / 3 = 2

라고 생각하게 되지만,

실제로 원래 배열에서는

[x x x x _ x x]

가 되어

4개 연속 + 2개 연속

뿐이므로

4/3 + 2/3 = 1

개의 꽃다발밖에 만들 수 없음.
## 정답
날짜를 이분탐색 하면서, 해당 날짜에 꽃다발을 몇개 만들 수 있는지를 검사한다.
