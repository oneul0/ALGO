# Q2. Sort an Array
## 정답
정렬 알고리즘 별 시간, 공간복잡도 비교

| 알고리즘       | 시간                      | 공간       | Stable |
| ---------- | ----------------------- | -------- | ------ |
| Merge Sort | O(n log n)              | O(n)     | O      |
| Quick Sort | 평균 O(n log n), 최악 O(n²) | O(log n) | X      |
| Heap Sort  | O(n log n)              | O(1)     | X      |

### Stable?
Stable은 안정 정렬(stable sort) 여부를 의미
`-> 값이 같은 원소들의 상대적인 순서가 정렬 후에도 유지되는가?`

| 값 | ID |
| - | -- |
| 5 | A  |
| 1 | B  |
| 3 | C  |
| 1 | D  |
| 2 | E  |

5A 1B 3C 1D 2E

`Stable Sort`
1B 1D 2E 3C 5A

`Unstable Sort`
1D 1B 2E 3C 5A
