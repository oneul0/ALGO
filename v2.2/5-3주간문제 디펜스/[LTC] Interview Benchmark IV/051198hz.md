# Special Binary String
## 접근
0과 1이 같은 개수이며, 개수만큼의 1이 접미사로 존재하는 문자열이 Special Binary String
아이디어를 떠올리지 못함.

## 풀이
Special Binary String은 
`1 + (n * Sub Special Binary String) + 0`
으로 구성돼 있으며, 따라서 Special Binary String이 사전순으로 뒤에 오려면
가운데의 `n * Sub Special Binary String` 이 사전순으로 뒤에 와야 함.
즉, Sub Special Binary String을 재배치해 사전순으로 앞에 오도록 해야 함.
